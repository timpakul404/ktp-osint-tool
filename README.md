
# KTP OSINT Tool

Tool untuk mengekstrak data KTP Indonesia menggunakan API OCR.

## Fitur
- Ekstrak data KTP dari foto (NIK, Nama, Alamat, TTL, dll)
- Parser NIK Indonesia
- Simpan hasil dalam format JSON dan TXT
- Support Termux & Python

## Cara Install

```bash
git clone https://github.com/timpakul404/ktp-osint-tool.git
cd ktp-osint-tool
pip install -r requirements.txt

cd ktp-osint-tool
python main.py
